# Discord Bank Bot

بوت بنك متكامل لدسكورد باللغة العربية مع نظام اقتصاد، ألعاب، وتفاعلات اجتماعية.

## المتطلبات
- Node.js v16.11.0 or higher
- Discord Bot Token

## التثبيت
1. قم بتحميل المستودع.
2. قم بتشغيل `npm install` لتثبيت المكتبات.
3. ضع توكن البوت الخاص بك في ملف `.env`.
4. قم بتشغيل البوت باستخدام `node index.js`.

## الأوامر
يمكنك رؤية جميع الأوامر بكتابة `!أوامر` أو `!help`.
- **راتب**: استلام الراتب اليومي.
- **رصيد**: معرفة رصيدك الحالي.
- **تحويل**: تحويل مبالغ لأعضاء آخرين.
- **توب**: قائمة أغنى الأعضاء.
- **نرد**: لعبة النرد للمراهنة.
- **فواكه**: لعبة الفواكه (Slots).
- **حظ**: تجربة حظك.
- **بروفايل**: عرض ملفك الشخصي.
- **بخشيش**: إعطاء مبالغ بسيطة للأصدقاء.
